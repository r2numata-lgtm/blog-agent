## 📋 プロジェクト基本情報

### プロジェクト名
**MyBlog AI**

### キャッチコピー
**「あなたの書き方で、AIが記事を書く。」**

### プロジェクトの目的
ClaudeとAWSを活用した、マルチプラットフォーム対応のブログ記事生成支援Webアプリケーション。ユーザーが入力した要点をもとに、チャットベースで高品質な記事を生成し、WordPress/Markdown/Notion/note 形式で出力する。

### ビジョン
- **短期**: 本格的な記事生成アプリの開発
- **中期**: ユーザーフィードバックによる改善・機能拡張
- **長期**: ブロガー必携のAIライティングツールへ

## 📝 記事生成フロー

### ステップ1: 入力画面
- 対象読者
- 記事の目的
- キーワード
- 本文の要点（10文字以上）
- 目標文字数
- 記事タイプ（情報提供/ハウツー/レビュー）
- **出力形式選択**（WordPress/Markdown/Notion/note）
- 「タイトル提案」ボタン

### ステップ2: タイトル選択画面
- AI生成のタイトル候補3案表示
- 各タイトルの狙い説明
- タイトル選択
- 「戻る」ボタン（入力画面へ）
- 「記事生成」ボタン

### ステップ3: 記事生成中
- ローディング表示
- 生成進捗表示

### ステップ4: 完了
- 記事一覧に自動追加
- エディタ画面が開く

### 戻るボタン
- 各ステップに「戻る」ボタンを配置
- 入力情報が失われる可能性がある場合は警告を表示

## 📄 出力形式

### 内部データ形式
**構造化JSON（中間フォーマット）**
```json
{
  "title": "記事タイトル",
  "blocks": [
    {
      "type": "heading",
      "level": 2,
      "content": "見出し"
    },
    {
      "type": "paragraph",
      "content": "本文テキスト"
    },
    {
      "type": "list",
      "ordered": false,
      "items": ["項目1", "項目2"]
    },
    {
      "type": "box",
      "boxType": "info",
      "content": "情報ボックス"
    },
    {
      "type": "balloon",
      "position": "left",
      "icon": "question",
      "content": "吹き出し"
    }
  ],
  "meta": {
    "description": "メタディスクリプション",
    "keywords": ["キーワード1", "キーワード2"],
    "slug": "article-slug"
  }
}
```

### 最終出力形式
| 形式 | 説明 |
|------|------|
| **WordPress** | Gutenbergブロックエディタ形式（HTMLコメント付き） |
| **Markdown** | 標準Markdown形式 |
| **Notion** | Notion互換Markdown |
| **note** | note.com用テキスト形式 |

## ✏️ エディタ画面

### アクセス方法
- 記事一覧から記事を選択して開く
- 記事生成完了後に自動的に開く

### タブ構成
1. **コード** - 選択された出力形式のソースコード表示・編集
2. **ブロックエディタ** - WordPress形式選択時のみ表示。Gutenbergライクなビジュアル編集
3. **プレビュー** - 記事のプレビュー表示

### チャット修正機能
- エディタ画面右側または下部にチャットパネル
- AIへの指示で記事を修正
- 差分表示で変更点を確認
- 変更の承認/却下

### WordPressブロック設定
- UI言語: 日本語

## 💳 料金プラン

### 無料プラン
- 月3記事まで生成可能
- 基本的な装飾のみ
- Markdown/note出力のみ

### スタンダードプラン
- 月30記事まで生成可能
- 全ての装飾利用可能
- 全ての出力形式対応
- チャット修正無制限

### プロプラン
- 無制限の記事生成
- 優先サポート
- API連携（将来）

### プランアップグレード画面
- プラン比較表
- 料金表示
- 決済機能（将来）

## 🛠️ 技術スタック概要

### フロントエンド
- **React** + **TypeScript**
- **Tailwind CSS**
- **Gutenbergブロックエディタライブラリ**
  - @wordpress/block-editor
  - @wordpress/blocks
  - @wordpress/components

### バックエンド
- **AWS Lambda** (Python 3.11)
- **API Gateway** (REST API)
- **DynamoDB**

### AI
- **Claude API** (Anthropic)
  - モデル: claude-sonnet-4-20250514
  - 会話履歴管理
  - ストリーミングレスポンス

### インフラ
- **AWS** (S3, CloudFront, Cognito, CloudWatch, etc.)
